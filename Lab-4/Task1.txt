Задание. 1. Создать класс «Зачетная Книжка» с внутренним классом, с помощью объектов которого можно хранить информацию о сессиях, зачетах, экзаменах.
2. Реализовать тестовую программу для проверки работоспособности приложения.
Информацию о зачетных книжках студентов хранить в коллекции.
Например, считывать из входного файла input.txtинформацию о студентах и записывать в выходной файл output.txt информацию о студентах – отличниках, сдавших все экзамены на оценки не ниже 9, а также сдавших все зачеты. В выходной файл записать следующую информацию о студенте: фамилия, имя, отчество, курс, группа, номер сессии (1-9), название предмета, оценка для экзамена.
3. Определить класс «Student» и атрибуты класса «Оценки, полученные студентами в сессию». По возможности использовать assert и исключения для обработки ошибочных ситуаций.
Необходимо реализовать методы определения:
- Среднего балла учебной группы в сессию (использовать класс Group);
- Средний балл студентов;
- Количество отличников;
- Количество студентов, имеющих «4».
